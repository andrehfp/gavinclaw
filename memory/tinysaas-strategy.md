## Análise de Competidores

### Landing Page Tools
**Unbounce, Leadpages, Instapage**
- *Fraqueza:* Muito caros, complex demais
- *Oportunidade:* Tool específico para SaaS simples
- *Posicionamento:* "Landing pages para indie hackers, não agencies"

### SEO Tools
**SEMrush, Ahrefs, Ubersuggest**
- *Fraqueza:* Preços altos, features demais
- *Oportunidade:* Audit focused em micro-SaaS essentials
- *Posicionamento:* "SEO sem bloat, só o que converte"

### AI Coding Tools
**GitHub Copilot, Cursor, Claude Code**
- *Fraqueza:* Contexto internacional, inglês only
- *Oportunidade:* Brazilian-specific context e português
- *Posicionamento:* "IA coding para o mercado brasileiro"

### Analytics Tools
**Google Analytics, Mixpanel, Amplitude**
- *Fraqueza:* Complexity overwhelming para small businesses
- *Oportunidade:* Metrics simples e actionable
- *Posicionamento:* "Analytics que indie hackers realmente usam"
