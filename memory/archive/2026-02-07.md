### Agent Commerce API (Innovative!)
- Bot-driven checkout - no web UI required
- Flow: Bot → API → Payment (PIX/Card link) → Webhook → Bot notified
- Endpoints designed:
  - POST /api/commerce/start
  - POST /api/commerce/select
  - POST /api/commerce/pay
  - GET /api/commerce/status/{session_id}
- Webhook to OpenClaw Gateway `/hooks/wake` - bot doesn't need own webhook!

- Using LemonSqueezy (not Stripe) - they handle tax compliance
- PIX for Brazil (instant, no card needed)
- Card via hosted checkout (secure, no PCI burden)
- No subscription - buy credits when needed
