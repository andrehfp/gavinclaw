# MoldaSpace Linear. Ritual semanal (15-20 min)

## Quando
- Toda segunda-feira de manhã (America/Sao_Paulo)

## 1) Limpeza rápida (3 min)
- Arquivar/cancelar tasks obsoletas
- Garantir que cada task tenha owner e due date

## 2) Planejamento da semana (7 min)
- Escolher 3-7 tasks para `This Week`
- Priorizar por impacto em receita/conversão
- Validar que cada task tem KPI esperado

## 3) Destravamento (5 min)
- Revisar coluna `Blocked`
- Definir próxima ação concreta para cada item
- Reatribuir owner se necessário

## 4) Fechamento operacional (3 min)
- Para cada `Done`, registrar resultado curto:
  - o que foi feito
  - KPI parcial/final
  - próximo passo (se houver)

## Regra de roteamento
- Dev/código: GitHub Issues
- Não-dev (ops, growth, conteúdo, SEO, comunidade, analytics, parcerias): Linear
