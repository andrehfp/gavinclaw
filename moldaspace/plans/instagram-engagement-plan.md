# Instagram Engagement Plan - @studio.maia.arch

## Account Age Phases

### Phase 1: Warm-up (Week 1 - Feb 15-21)
- **Likes:** 5-10/day
- **Comments:** 2-3/day
- **Follows:** 1-2/day
- **Our posts:** 1/day (already scheduled)
- **DMs:** None
- **Stories:** None yet

### Phase 2: Growth (Week 2 - Feb 22-28)
- **Likes:** 15-20/day
- **Comments:** 5-8/day
- **Follows:** 3-5/day
- **Our posts:** 2/day
- **DMs:** Reply only
- **Stories:** Start sharing (behind-the-scenes renders)

### Phase 3: Acceleration (Week 3-4 - Mar 1-14)
- **Likes:** 30-40/day
- **Comments:** 10-15/day
- **Follows:** 5-10/day
- **Our posts:** 2/day + Stories
- **DMs:** Engage with bigger accounts
- **Collabs:** Start reaching out

### Phase 4: Cruise (Mar 15+)
- **Likes:** 40-50/day
- **Comments:** 15-20/day
- **Follows:** Strategic only
- **Content:** 2 posts/day + daily Stories + Reels
- **Collabs:** Active partnerships

## Safety Rules
- MAX 60 actions/hour (Instagram hard limit)
- 15-30 second gap between actions
- Vary timing (don't run at exact same time daily)
- If captcha or "try again later" → STOP 24h, reduce next day by 50%
- If shadowban suspected (reach drops to near-zero) → STOP 48h, content-only for a week
- Comments MUST be specific to the post (never generic)
- No hashtag spam in comments
- Follow/unfollow ratio: keep follows, don't unfollow in first month
