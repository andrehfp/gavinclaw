# SKILL: Keywords Everywhere via MCP

## Objetivo
Descobrir oportunidades de keyword com volume, CPC e termos relacionados para alimentar o funil SEO.

## Inputs
- Seed terms (ex: lumion alternative, ai render sketchup)
- País/idioma alvo (EN global + PT-BR)

## Output
- Cluster de keywords priorizadas com:
  - volume
  - CPC
  - competição
  - intenção (TOFU/MOFU/BOFU)
  - prioridade (alta/média/baixa)

## Workflow
1. Expandir seed terms em related queries.
2. Classificar por intenção.
3. Eliminar termos irrelevantes/sem fit produto.
4. Entregar shortlist semanal (top 10) para páginas novas.

## Regras de priorização
- Primeiro BOFU (alternative/vs/for tool)
- Depois MOFU (best tools/workflow)
- TOFU só para suportar autoridade
