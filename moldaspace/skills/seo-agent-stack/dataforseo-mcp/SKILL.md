# SKILL: DataForSEO via MCP

## Objetivo
Tomar decisão por dados de SERP reais e monitorar impacto das páginas publicadas.

## Inputs
- Lista de keywords priorizadas
- URLs alvo da MoldaSpace
- Concorrentes diretos por cluster

## Output
- Radar de SERP com:
  - dificuldade relativa
  - tipo de resultado dominante (listicle/comparison/tool page)
  - lacunas de conteúdo
  - tracking de posição por URL

## Rotina
1. Pré-publicação:
   - validar se keyword vale disputa
   - mapear top pages concorrentes
2. Pós-publicação:
   - monitorar ranking e mudanças semanais
   - sinalizar páginas para refresh

## Alertas
- Página com impression alta e CTR baixa
- Queda de posição por 2 semanas
- Canibalização entre URLs internas
