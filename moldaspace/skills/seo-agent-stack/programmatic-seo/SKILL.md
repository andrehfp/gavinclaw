# SKILL: Programmatic SEO (MoldaSpace)

## Objetivo
Criar e manter páginas transacionais de alta conversão para MoldaSpace sem cair em thin content.

## Inputs
- Keyword alvo e intenção (TOFU/MOFU/BOFU)
- Dados de SERP (DataForSEO)
- Dados de volume/CPC/relacionadas (Keywords Everywhere)
- Provas visuais (renders, before/after, casos)

## Output
- Página pronta com:
  - H1 orientado a intenção
  - comparação útil e honesta
  - seção “quando usar MoldaSpace”
  - prova visual
  - CTA forte: “5 free renders”

## Template mínimo por página
1. Dor principal do usuário
2. Comparação objetiva (sem hype fake)
3. Como MoldaSpace resolve
4. Before/after real
5. FAQ curta
6. CTA

## Regras
- Não publicar se conteúdo estiver genérico.
- Não duplicar intenção já coberta por outra URL.
- Sempre incluir ângulo específico (tool/niche/use-case).

## Frequência recomendada
- 2-3 páginas/semana no começo.
- Revisar páginas antigas toda semana por CTR e ranking.
