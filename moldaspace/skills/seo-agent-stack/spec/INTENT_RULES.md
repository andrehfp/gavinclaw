# Intent Rules

## BOFU (alta prioridade)
Sinais:
- alternative
- vs
- best for <tool/use-case>
- pricing/comparison

## MOFU
Sinais:
- best tools
- workflow
- how to choose

## TOFU
Sinais:
- what is
- guide
- tips

## Priorização
1. BOFU
2. MOFU
3. TOFU

## Exceções
- TOFU pode subir prioridade se volume alto + baixa competição + fit visual claro.
