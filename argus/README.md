# Argus

Projeto: plataforma web-first de IA para agências e times de conteúdo.

## Tese atual
Argus não é "mais uma ferramenta de social media".

**Counter-positioning:**
- "Somos a camada de aprovação entre IA e cliente."
- "Antes de publicar, passa no Argus."

## MVP atual (v1)
1. Chat (hub operacional)
2. Geração de campanha (variações por canal)
3. Aprovação do cliente por link (approve/reject + comentário)

## Estrutura
- `MISSION_CONTROL.md` → direção e prioridades
- `memory/` → memória de decisões, learnings e diário
- `plans/` → planos de produto/GTM
- `content/` → copy, landing, outreach
- `dashboard/` → artefatos de acompanhamento
- `skills/` → automações/agents específicos do projeto
